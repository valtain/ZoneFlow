# Test Cases

- ExplorationMode 진입 시 HUD가 슬라이드인 연출로 나타난다
- ExplorationMode 퇴장 시 HUD가 슬라이드아웃 연출로 사라진다
- BattleMode 등 다른 모드 전환(Sleep) 시 HUD가 숨겨진다
- BattleMode에서 ExplorationMode로 복귀(Resume) 시 HUD가 다시 나타난다
- HUD에 현재 Zone 이름이 표시된다
- HP 변경 시 HUD 체력바가 즉시 반영된다
- ExplorationMode 완전 종료(Stopped) 후 HUD 인스턴스가 파괴된다
